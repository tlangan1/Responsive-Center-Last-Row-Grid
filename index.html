Center the last row of grid having a fixed number of columns using CSS only.

This approach require that the actual number of columns be the factorial of the number of "visible" columns. For example, a 3 column grid will require 6 columns, a 4 column grid will require 24 columns, a 5 column grid will require 120 columns and so on. As such, this approach is not scalable to grids with a large number of columns.

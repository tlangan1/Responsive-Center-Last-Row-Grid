Both of these approaches require that the actual number of columns be the factorial of the number of columns visibly displayed. For example, a 3 column grid will require 6 columns, a 4 column grid will require 24 columns, a 5 column grid will require 120 columns and so on. As such, these approaches are not scalable. also, grid-gap would be exceedingly difficult to implement.
